Abo anfragen

Häufige Fragen

Alles rund um Bestellung, Lieferung und unsere Dienstleistungen.

Wie funktioniert die Lieferung?
Wir liefern Montag bis Samstag innerhalb der Stadt Zürich. Die Lieferung kostet CHF 12 — ab einem Bestellwert von CHF 60 ist sie gratis. Bestellungen bis 14:00 Uhr werden am selben Tag geliefert.
Kann ich einen individuellen Strauss bestellen?
Selbstverständlich! Rufen Sie uns an oder schreiben Sie uns eine E-Mail mit Ihren Wünschen zu Farben, Blumenarten und Anlass. Wir beraten Sie gerne persönlich.
Bieten Sie Hochzeitsfloristik an?
Ja, Hochzeitsfloristik ist eine unserer Spezialitäten. Vom Brautstrauss über Tischdekorationen bis hin zum Kirchenschmuck — vereinbaren Sie frühzeitig ein unverbindliches Beratungsgespräch.
Wie funktioniert das Blumen-Abo?
Sie wählen die Lieferfrequenz (wöchentlich oder alle 2 Wochen) und die Strauss-Grösse. Wir liefern Ihnen einen frisch gebundenen, saisonalen Strauss direkt nach Hause. Das Abo ist jederzeit pausierbar und kündbar.
Welche Zahlungsmethoden akzeptieren Sie?
Im Laden: Bargeld, EC-Karte und Kreditkarte. Online: Kreditkarte, TWINT oder auf Rechnung. Für Hochzeiten und Firmenservice erstellen wir individuelle Offerten.

Unsere Philosophie

Zahnmedizin, die auf Vertrauen und Kompetenz baut

In unserer Praxis an der Seestrasse in Zürich Enge steht der Mensch im Mittelpunkt. Wir glauben daran, dass gute Zahnmedizin mehr ist als technische Perfektion – sie erfordert Einfühlungsvermögen, Transparenz und echte Beziehung.

„Jeder Patient verdient eine Behandlung, die auf seine individuellen Bedürfnisse zugeschnitten ist. Wir nehmen uns die Zeit, die es braucht.“

Unser Angebot umfasst die gesamte Bandbreite moderner Zahnmedizin – von Prophylaxe und ästhetischer Zahnheilkunde bis zu Implantologie und Endodontie. Wir arbeiten mit den neusten digitalen Technologien: 3D-Röntgen, Intraoral-Scanner und computergestützte Diagnostik.

Auch Angstpatienten finden bei uns einen stressfreien Zugang zur Behandlung. Wir erklären jeden Schritt und geben Ihnen die Kontrolle über Ihre Behandlung.

Klick auf „Jetzt starten" oben rechts. Du füllst ein kurzes Formular aus — Firmenname, E-Mail-Adresse, Öffnungszeiten und optional deine Preise. Das dauert ca. 2 Minuten.

Zum Anmeldeformular
2

Wir richten deinen Bot ein

Nach deiner Anmeldung konfigurieren wir deinen persönlichen Chatbot. Er wird mit deinen Öffnungszeiten, Preisen und dem von dir gewählten Tonfall eingerichtet. Du musst nichts weiter tun.

Dein Bot wird innerhalb von 24 Stunden aktiviert.
3

Du erhältst deinen persönlichen Code per E-Mail

Du bekommst eine E-Mail mit einer einzelnen Zeile Code. Das ist dein Chatbot — einzigartig für dein Business. Er sieht ungefähr so aus:

<script src="https://chatbot-prod-v1.vercel.app/widget.js" data-client-id="dein-business-name"></script>

Kopiere diesen Code — du brauchst ihn im nächsten Schritt.

4

Code auf deiner Website einfügen

Der Code muss einmal auf deiner Website eingefügt werden — kurz vor dem </body> Tag. Je nach System geht das so:

WordPress

Geh zu Design → Theme-Editor → footer.php. Oder installiere das Plugin „Insert Headers and Footers". Dort den Code im Bereich „Footer Scripts" einfügen und speichern.

Wix

Geh zu Einstellungen → Erweitert → Benutzerdefinierten Code. Klick auf „+ Code hinzufügen", wähle „Körper — Ende", füge den Code ein und speichere.

Squarespace

Geh zu Einstellungen → Erweitert → Code-Einschleusung. Füge den Code im Bereich „Footer" ein und speichere die Änderungen.

Eigene Website

Öffne deine index.html im Editor. Suche nach </body> am Ende der Datei. Füge den Code direkt davor ein und speichere.

Nicht sicher wo das ist? Kein Problem — schreib uns einfach auf info@botswiss.ch. Wir fügen den Code für dich ein. Das ist im Preis inbegriffen.
5

Dein Chatbot ist live

Sobald der Code eingefügt ist, erscheint der Chat-Button unten rechts auf deiner Website. Deine Kunden können sofort Fragen stellen, Öffnungszeiten erfragen und Reservierungen machen — und du bekommst bei jeder Reservierung automatisch eine E-Mail.

Fertig. Der Bot läuft ab sofort 24/7 auf deiner Website.
